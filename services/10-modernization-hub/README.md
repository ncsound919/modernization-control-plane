# Module 10 — Modernization Hub

## Purpose
The **unified command center** for the entire Modernization Control Plane. Surfaces debt heatmaps, certificate health, migration runbooks, agent workflow composition, governance controls, and sustainability KPIs in a single, opinionated enterprise UI. The commercial layer where outcome-based pricing is enforced.

## Views and Features

### Debt Heatmap
- Visual representation of technical debt graph from Module 01
- Risk scoring by service, team, and domain
- Drill-down to individual systems, certs, and legacy dependencies
- "Modernization readiness" score per subsystem

### Certificate Health Dashboard
- Real-time status of all discovered certs (red/amber/green)
- Days to expiry, renewal status, CA, and owner
- One-click "Enroll in automated renewal" via CLM Module 03
- Compliance status for 200-day and 47-day validity thresholds

### Migration Runbook Builder
- Step-by-step migration plans generated by Migration Agent
- Per-vertical templates (banking, healthcare)
- Tier 1 / Tier 2 data split configuration
- Progress tracking and rollback checkpoints

### Agent Workflow Composer
- Visual DAG builder for multi-agent modernization workflows
- Drag-and-drop agent assignment
- SLO and policy constraint configuration
- Live execution monitoring with audit log feed

### Governance Console
- Policy status and compliance posture per tenant
- Kill switch controls (tenant-level and workflow-level)
- Audit log search and export
- Violation alerts and remediation tracking

### Sustainability KPI Board
- gCO2e dashboard (per call, per workflow, per project)
- Carbon budget status and trend
- Green region utilization
- ESG report generation

### Outcome Pricing Dashboard
- Value metrics per customer: certs auto-renewed, legacy debt removed, time-to-ship improvement
- Billing tied to verified outcomes, not seat licenses
- ROI calculator comparing legacy baseline to modernized state

## Pilot Launch (Summer 2026)
- Controlled rollout: 1–5% of a design-partner's user base
- Banking vertical first: cert health + sidecar wrapping + one agent workflow
- Measure: feature ship time, cert outages prevented, compliance wins

## Tech Stack
- **Framework:** Next.js 15 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS + shadcn/ui
- **API Layer:** tRPC (type-safe client-server)
- **Auth:** NextAuth.js (SSO, SAML for enterprise)
- **Charts:** Recharts / Observable Plot
- **State:** Zustand + React Query
